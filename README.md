🛒 DailyDrop – Grocery & Essentials Frontend UI

DailyDrop is a frontend-only e-commerce web UI inspired by fast grocery delivery platforms like Swiggy Instamart / Amazon Fresh / Blinkit.
This project focuses on UI design, layout consistency, responsiveness, and user experience, built using HTML, CSS, and JavaScript.

⸻

📌 Project Objective
	•	Build a clean and modern grocery delivery website UI
	•	Practice real-world frontend concepts
	•	Maintain consistent image sizes and layouts
	•	Implement basic interactivity using JavaScript
	•	Ensure responsiveness across multiple screen sizes

⸻

🧩 Features

🏠 Home Page
	•	Hero image slider with navigation controls
	•	Category cards (Fruits, Dairy, Snacks, Personal Care, etc.)
	•	Horizontal product sliders
	•	Deals and offers section

📄 Product Page
	•	Product image gallery
	•	Product details & pricing
	•	Color/variant selection
	•	“About this item” section
	•	Purchase summary card
	•	Related deals slider

🛒 Cart Page
	•	Cart item listing
	•	Quantity selector
	•	Price summary
	•	Delivery information
	•	Checkout section

🔐 Authentication Pages
	•	Login page
	•	Signup page
	•	Clean and minimal UI

⸻

🛠️ Tech Stack Used
	•	HTML5 – Page structure
	•	CSS3 – Styling & responsiveness
	•	JavaScript (Vanilla JS) –
	•	Image slider
	•	Horizontal scroll using mouse wheel

⸻

📱 Responsiveness
	•	Designed using CSS Flexbox
	•	Media queries applied for:
	•	Desktop
	•	Laptop
	•	Tablet
	•	Mobile
	•	Layout adapts without breaking text or image proportions
